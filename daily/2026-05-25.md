## 我能解释的概念

### 1. 为什么先加载 base，再替换 LoRA

base checkpoint 里的参数名是普通 MiniGPT 的名字：

```text
blocks.0.attn.qkv.weight
blocks.0.attn.qkv.bias
```

替换成 LoRA 后，参数名会变成：

```text
blocks.0.attn.qkv.linear.weight
blocks.0.attn.qkv.linear.bias
blocks.0.attn.qkv.lora_A.weight
blocks.0.attn.qkv.lora_B.weight
```

所以正确顺序是：

```python
model = MiniGPT(**checkpoint["config"])
model.load_state_dict(checkpoint["model"])
model = replace_qkv_with_lora(model, r=8, alpha=16)
```

### 2. optimizer 只拿可训练参数

```python
trainable_params = [
    p
    for p in model.parameters()
    if p.requires_grad
]
optimizer = torch.optim.AdamW(trainable_params, lr=3e-4)
```

这表示 base 参数参与 forward，但不计算梯度、不被 optimizer 更新。

### 3. best adapter 保存的是什么

正式训练 LoRA 时，best checkpoint 不保存整个 MiniGPT，而是只保存 adapter：

```python
lora_state = {
    name: tensor
    for name, tensor in model.state_dict().items()
    if "lora_" in name
}
torch.save(lora_state, adapter_path)
```

保存后的 adapter 需要和对应 base checkpoint 一起使用。

## 今天的关键测试

- `build_lora_model_from_checkpoint` 会先加载 base 权重，再替换 qkv LoRA。
- `make_lora_optimizer` 只包含 `requires_grad=True` 的 LoRA 参数。
- `train_lora_adapter` 短训练后会保存 adapter。
- 保存出来的 adapter 只包含 `lora_` 参数。
- `r=8, dim=128, num_layers=4` 时，trainable parameters 是 `16384`。

## 今天要手撕复现的内容

```python
def get_trainable_parameters(model):
    return [p for p in model.parameters() if p.requires_grad]
```

```python
def make_lora_optimizer(model, learning_rate):
    return torch.optim.AdamW(
        get_trainable_parameters(model),
        lr=learning_rate,
    )
```

```python
def build_lora_model_from_checkpoint(checkpoint_path, r=8, alpha=16):
    checkpoint = torch.load(checkpoint_path, map_location="cpu")
    model = MiniGPT(**checkpoint["config"])
    model.load_state_dict(checkpoint["model"])
    model = replace_qkv_with_lora(model, r=r, alpha=alpha)
    return model, checkpoint
```

## 今天的误区提醒

- `strict=False` 用在加载 adapter，不用在加载完整 base checkpoint。
- `requires_grad=False` 不代表参数不参与 forward。
- adapter 不是完整模型，不能单独 generate。
- validation 只评估，不更新参数，所以要 `model.eval()` 和 `torch.no_grad()`。
